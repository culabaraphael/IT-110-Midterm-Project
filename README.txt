PROJECT DESCRIPTION:
------------------------------------------
The Weather Checker Web Application allows users to search for real-time weather information in any city worldwide using the OpenWeatherMap API. It demonstrates knowledge in HTML, CSS, and JavaScript—particularly API fetching using async/await, DOM manipulation, and user-friendly error handling.

------------------------------------------
FEATURES:
------------------------------------------
✅ Displays live weather data including:
   • Temperature (°C)
   • Weather Condition
   • Humidity
   • Wind Speed
   • Visibility
   • Pressure
   • Sunrise and Sunset times

✅ Error Handling:
   • Displays “No internet connection” message when offline.
   • Handles invalid city input with a friendly message.
   • Displays a clear “Could not fetch data” message for API request failures.

✅ Dark/Light Mode Toggle:
   • Allows users to switch between light and dark themes.

✅ Responsive Design:
   • Fully responsive layout that adapts to mobile and desktop screens.

✅ Interactive Portfolio:
   • Includes team introduction, project overview, and a link to the Weather Checker App.

HOW TO RUN THE PROJECT:
------------------------------------------
1. Ensure you have an active internet connection.
2. Extract the ZIP file and open the project folder.
3. Open the file named `index.html` in your browser.
4. From the portfolio page, click “View Weather App” to access the weather checker.
5. Enter any city (e.g., London, Tokyo, Manila) and click “Search”.
